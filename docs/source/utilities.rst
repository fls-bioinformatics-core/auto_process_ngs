.. _utilities_run_qc:

``run_qc.py``: run QC standalone
********************************

The ``run_qc.py`` utility allows the QC pipeline to be run standalone,
outside of the ``auto_process`` context.

For example::

    run_qc.py ./my_project

Use ``run_qc.py --help`` to see the available options.
